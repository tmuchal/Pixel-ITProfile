## 사용법

### 1단계 — Vercel에 배포하기

위 **Deploy** 버튼을 클릭하면 Vercel 사이트로 이동해.
GitHub 계정으로 로그인하고 배포를 완료하면 내 고유 주소가 생겨.
예: `https://pixel-itprofile-홍길동.vercel.app`

### 2단계 — 내 GitHub 프로필에 추가하기

GitHub에서 내 이름과 같은 레포 (예: `홍길동/홍길동`)의 `README.md`를 열고,
아래 코드를 복붙해. **주소 부분만 1단계에서 받은 내 주소로 바꾸면 끝.**

```markdown
![Pixel IT Profile](https://여기에-내-vercel-주소/api/cat?scene=dubai&theme=cyberpunk)
```

씬을 바꾸고 싶으면 `scene=dubai` 부분을 `scene=paris` 또는 `scene=italy`로 교체.

## 파라미터

| 파라미터 | 옵션 | 기본값 |
|---------|------|--------|
| `scene` | `dubai` `italy` `paris` | `paris` |
| `theme` | `matrix` `cyberpunk` `synthwave` `ocean` `nord` `dracula` `tokyonight` `monokai` `hacker` `neon` `terminal` `retro` | `matrix` |
| `width` | 최대 `1200` | `800` |
| `height` | 최대 `300` | `150` |
