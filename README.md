## 사용법

**1. Vercel 배포** — 위 Deploy 버튼 클릭

**2. 내 README에 추가**

```markdown
![Pixel IT Profile](https://your-app.vercel.app/api/cat?scene=dubai&theme=cyberpunk)
```

`your-app.vercel.app`을 배포된 내 Vercel 주소로, `scene`을 원하는 씬으로 교체.
